# chronic-disease-prediction
##  Overview
This project uses Machine Learning to predict high-risk chronic disease cases using public health data.

## Objective
To classify whether a person is at high risk of chronic disease based on indicators.

## Dataset
- Source: U.S. Chronic Disease Indicators (data.gov)
- Contains health-related indicators such as gender, location, and disease values.

## Machine Learning Models
- Logistic Regression
- Neural Network

## Features
- Data cleaning and preprocessing
- Data visualization
- Model training and evaluation

## Tools Used
- Python
- Google Colab
- Pandas, Scikit-learn, TensorFlow

## How to Run
1. Open the notebook in Google Colab
2. Run all cells step by step
